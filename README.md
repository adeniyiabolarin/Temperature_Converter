# Temperature Converter

A lightweight temperature converter application that performs quick and accurate conversions between Celsius and Fahrenheit.

## Features

- Convert Celsius to Fahrenheit instantly
- Convert Fahrenheit to Celsius instantly
- Simple and user-friendly interface
- Fast and accurate temperature calculations
- Responsive design for different screen sizes

## Built With

- HTML
- CSS
- JavaScript

## How It Works

Users can enter a temperature value and convert it between Celsius and Fahrenheit with a single click. The application provides immediate and accurate conversion results.

## Installation

1. Clone the repository

```bash
git clone <your-repository-link>
```

2. Navigate to the project folder

```bash
cd temperature-converter
```

3. Open the `index.html` file in your browser
